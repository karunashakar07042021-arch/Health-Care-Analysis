# Health-Care-Analysis
# Project Description:
This project focuses on analyzing health care data to identify trends in patient visits, treatment outcomes, and resource utilization. Using data cleaning, exploratory analysis, and interactive dashboards, the project aims to support data-driven decision-making and improve operational efficiency in the health care sector.
# Key Features:

Cleaned and transformed raw health care data for analysis

Analyzed patient demographics, admission trends, and treatment patterns

Built interactive dashboards to visualize key health metrics

Generated insights to support better planning and patient care
# Tools & Technologies:
Excel, Power Query
